##  Installation

> For the first __Installation :__ 
 
 - ___Linux__ & __macOS___ : 
   - run in the __Terminal__ :
  
```bash
    cd ~ && 
    wget https://github.com/sehzam/ali/archive/refs/heads/main.zip && 
    unzip main.zip && 
    rm main.zip && 
    . ./ali-main/.setup
```
